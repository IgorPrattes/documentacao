Seção: Como identificar o ID da pasta pai e do documento filho

Nesta seção, vamos explicar como identificar o ID da pasta pai e do documento filho em uma listagem de documentos. Isso é importante para que você saiba como utilizar os IDs corretamente ao consultar os endpoints da API.

Pasta Pai

Na imagem abaixo, o ícone da pasta indica que é uma pasta pai. O código que aparece na coluna "Código" é o ID da pasta pai. Este ID será usado no endpoint para realizar consultas relacionadas a essa pasta.

Acessar Pasta

Documento Filho

Quando você navega para dentro de uma pasta pai, encontrará os documentos filhos. Eles têm um ícone diferente e um código distinto. Este código será utilizado para acessar dados específicos desse documento.
